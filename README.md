# HerHub

> always near her.  
> 캠퍼스 월경 긴급 상황을 위한 스테이션 및 SOS 연결 서비스

## Overview

HerHub는 대학 캠퍼스에서 예상치 못한 월경 상황을 겪는 사용자가 빠르게 여성용품을 구하고 도움을 요청할 수 있도록 돕는 캠퍼스 기반 월경 긴급 대응 서비스입니다.

대학생의 캠퍼스 생활은 야외 이동을 많이 포함하지만, 여성용품 공급처는 대부분 실내 공간에 한정되어 있습니다. 또한 월경과 여성용품에 대한 타부 인식으로 인해 긴급 상황에서 도움을 요청하는 데 심리적 부담이 발생합니다.

HerHub는 캠퍼스 내 스테이션과 모바일 기반 SOS 기능을 통해 즉각적이고 확실한 대응 인프라를 제공합니다.

## Problem

캠퍼스에서 예상치 못한 월경 상황이 발생했을 때, 사용자는 다음과 같은 어려움을 겪습니다.

- 가까운 여성용품 공급처를 찾기 어려움
- 건물 밖 또는 이동 중일 때 즉각적으로 대응하기 어려움
- 비치함 재고 여부를 사전에 알 수 없음
- 편의점 또는 매점까지 이동해야 하는 부담이 있음
- 옷 오염 등으로 이동 자체가 꺼려질 수 있음
- 주변 사람에게 도움을 요청하는 것에 심리적 부담이 있음

이 문제는 개인의 준비 부족이 아니라 캠퍼스 인프라의 구조적 공백에 가깝습니다.

## Solution

HerHub는 캠퍼스 내 여성용품 스테이션과 모바일 SOS 연결 기능을 결합합니다.

사용자는 모바일 앱 또는 웹을 통해 가장 가까운 스테이션 위치와 재고를 확인하고, 필요 시 다른 사용자에게 긴급 도움을 요청할 수 있습니다. 도움을 제공하는 사용자는 SOS 요청을 수락하고, 보상을 통해 나눔에 참여할 수 있습니다.

## Key Features

### 1. 스테이션 위치 확인

사용자는 지도에서 가장 가까운 여성용품 스테이션을 확인할 수 있습니다.

- 내 위치 기반 스테이션 탐색
- 스테이션 위치 표시
- 거리 정보 제공
- QR 기반 이용 흐름

### 2. 재고 확인

스테이션에 필요한 여성용품이 남아 있는지 확인할 수 있습니다.

- 실시간 또는 최신 재고 정보 표시
- 제품 종류 확인
- 품절 상황 사전 인지
- 불필요한 이동 감소

### 3. 여성용품 구매

사용자는 스테이션에서 필요한 여성용품을 바로 구매할 수 있습니다.

- QR 인식
- 제품 선택
- 결제
- 수령 확인

### 4. SOS 요청

스테이션 이용이 어렵거나 긴급한 경우, 주변 사용자에게 SOS를 보낼 수 있습니다.

- 필요한 용품 선택
- 긴급 요청 전송
- 위치 또는 만남 장소 설정
- 요청 상태 확인

### 5. SOS 수락

도움을 줄 수 있는 사용자는 SOS 요청을 수락하고 전달을 진행할 수 있습니다.

- 요청 확인
- 수락 또는 거절
- 거래/전달 진행
- 보상 지급
- 완료 내역 확인

## Target Users

- 캠퍼스에서 월경 긴급 상황을 경험한 학생
- 월경 주기가 불규칙한 사용자
- 여성용품을 챙기는 것을 종종 잊는 사용자
- 야외 이동이 많은 대학생
- 주변에 도움을 요청하는 것이 부담스러운 사용자
- 월경 관련 인프라가 부족한 캠퍼스 구성원

## User Research

HerHub는 설문조사와 반구조화 인터뷰를 통해 문제 상황을 검증했습니다.

리서치 결과, 많은 사용자가 최근 캠퍼스 야외 공간에서 예상치 못한 월경 상황을 경험했으며, 불편함, 짜증, 불안감, 당혹감 등의 감정을 느꼈습니다.

기존 대처 방식은 편의점/매점 방문, 건물 화장실 이동, 친구에게 도움 요청, 참고 기다리기 등으로 나타났습니다. 이 과정에서 이동 시간, 비용 부담, 옷 오염으로 인한 이동 부담, 도움 요청의 어려움이 주요 페인포인트로 확인되었습니다.

## UX Goals

HerHub는 다음 세 가지 가치를 중심으로 설계되었습니다.

### 즉각성

긴급 상황에서 빠르게 대응할 수 있어야 합니다.

### 확실성

가까운 위치와 재고 여부를 확인해 불필요한 이동을 줄여야 합니다.

### 연결성

개인이 혼자 해결하지 못하는 상황에서 주변 사용자와 안전하게 연결될 수 있어야 합니다.
